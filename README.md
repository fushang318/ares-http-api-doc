# ares http api doc


## 本地运行编辑环境
```
npm install -g swagger
git clone https://github.com/fushang318/ares-http-api-doc.git
cd ares-http-api-doc
npm install
swagger project edit -s --host 0 -p 8080

# 浏览器打开 http://ip:8080 进行编辑，编辑会实时保存在 api/swagger/swagger.yaml
```
